

## 1. 格式

> ls \[-选项\] \[文件或目录路径\]

## 2. 参数 

```bash
-l 以长格式来显示文件的详细信息 use a long listding format
-a 列出所有项,包括隐藏文件 all 
-d 仅列出目录本身的信息,而非目录中文件列表信息 --正常情况下是全部列出的,包括目录和文件(目录和文件不是同一种概念)
-R 递归列出 # 同时列出字母下的所有文件 <2020-06-24 三 00:08>
-t 按时间进行列出(将文件依建立时间之先后次序列出)
-m 要以逗号分隔文件列表 
-h 文件大小显示为常见大小单位 B KB MB
-1 要强制每行一个文件 (这是数字 1，而不是小写的 L)
-A 要显示几乎所有隐藏文件 (. 和 .. 除外) 
-B 要隐藏备份文件，请使用 -B 或 --ignore-backups 隐藏常用备份格式 (此选项在 BSD 的 ls 中不可用) 
-s 倒着排序 reverse 
-S 以文件巨细排序。 # 按文件大小进行排序,从大到小进行排列。同时从小到大进行排列是附加一个-s参数就可以了。
```

### 3. 应用:

``` shell
ls -l | sort -n -r -k 5   # 注:按文件大小进行排序  # $ ls -l -S (正着)    $ls -l -S -s (反着)
ls s*                     # 注:列出以s开头的文件
ls *钢琴*                  # 列出所有带有钢琴字样的文件
ls *钢琴*  | wc -l         # 统计所有带有钢琴字样的文件
ls *.txt                  # 列出当前目录中所有的txt类型(以.txt结尾的文件) --偶尔会失效 
ls ./*.txt                # 列出当前目录中所有的txt类型(以.txt结尾的文件) ---一定可以  # 其实两个格式只是输出文件的格式不同，而其它方面基本上相同，不知为何前者偶尔会无法使用。
ls -d -a -l .             # 列出当前目录的详细信息   <2020-05-29 五 13:43>
ls *.c
ls -l | grep '^d'  # 列出所有的目录文件
ls -l | grep '^-'  # 列出所有的普通文件
```

配置别名

```bash
alias ls='ls -A -F -B --human --color' 
# 这一行导致你的 Bash shell 将 ls 命令解释为 ls -A -F -B --human --color。
alias ll='ls -l' 
alias la='ls -A' 
alias lh='ls -h'
```
